ang2rot

PURPOSE

Convert Euler angles to rotation matrix.

SYNOPSIS

function A = ang2rot(phi, theta, psi)

DESCRIPTION 

 Convert Euler angles to rotation matrix.
 Euler angles use the x-convention, i.e. phi, theta and psi are taken around the Z, X
 and again the Z-axis.

 See also: rot2ang
