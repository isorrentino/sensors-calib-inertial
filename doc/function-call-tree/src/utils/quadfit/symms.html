symms

PURPOSE

Create a symmetric matrix of symbolic variables.

SYNOPSIS

function S = symms(varargin)

DESCRIPTION 

 Create a symmetric matrix of symbolic variables.

 See also: symm

 Examples:
 A = symms('A',3)
    creates a symmetric symbolic 3-by-3 matrix
 A = symms('A',3,'positive')
    creates a symmetric symbolic 3-by-3 matrix with positive entries
