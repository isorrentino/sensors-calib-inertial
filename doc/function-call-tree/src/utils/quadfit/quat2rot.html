quat2rot

PURPOSE

Convert quaternion to rotation matrix.

SYNOPSIS

DESCRIPTION 

 Convert quaternion to rotation matrix.

 Input arguments:
 q:
    the rotation as a quaternion where the last entry is the "scalar term"

 Output arguments:
 R:
    a rotation matrix (i.e. an orthogonal matrix with determinant 1)
