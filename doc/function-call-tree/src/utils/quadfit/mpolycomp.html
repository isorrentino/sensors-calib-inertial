mpolycomp

PURPOSE

Matrix polynomial companion form.

SYNOPSIS

function [A,B] = mpolycomp(T)

DESCRIPTION 

 Matrix polynomial companion form.
 Builds two p*n-by-p*n matrices such that
 A = [T0   0   0   0]   B = [-T1 -T2 -T3 -T4]
     [ 0   I   0   0]       [  I   0   0   0]
     [ 0   0   I   0]       [  0   I   0   0]
     [ 0   0   0   I]       [  0   0   I   0]

 See also: polyeig, eig
