symmd

PURPOSE

Create a diagonal matrix of symbolic variables.

SYNOPSIS

function S = symmd(varargin)

DESCRIPTION 

 Create a diagonal matrix of symbolic variables.

 See also: symm

 Examples:
 D = symmd('D',3)
    creates a symbolic diagonal 3-by-3 matrix
 D = symmd('D',5,6)
    creates a symbolic diagonal 5-by-6 matrix
 D = symmd('D',[7,12])
    creates a symbolic diagonal 7-by-12 matrix
