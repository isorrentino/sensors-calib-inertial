imconictranslation

PURPOSE

Translation vector of a conic section given as an implicit equation.

SYNOPSIS

function t = imconictranslation(p)

DESCRIPTION 

 Translation vector of a conic section given as an implicit equation.
 The translation vector t maps a (possibly rotated) conic section in the
 origin to the location as defined by p.

 Input arguments:
 p:
    parameters of the conic section implicit equation p = [a b c d f g]

 Output arguments:
 t:
    translation (column) vector

 See also: imconictranslate, imconicrotation

CROSS-REFERENCE INFORMATION

This function calls:
  • imconicdiscr Discriminant for conic section.
  • imconicrotate Rotates a conic section given as an implicit equation.
  • imconicrotation Rotation matrix of a conic section given as an implicit equation.
This function is called by:

  • imconictranslate Translates a conic section given as an implicit equation.
