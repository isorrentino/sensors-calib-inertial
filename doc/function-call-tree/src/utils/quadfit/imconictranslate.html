imconictranslate

PURPOSE

Translates a conic section given as an implicit equation.

SYNOPSIS

function pt = imconictranslate(p, t)

DESCRIPTION 

 Translates a conic section given as an implicit equation.

 Input arguments:
 p:
    the parameter vector p = [a b c d f g]
 t:
    the translation vector t = [x y]

 See also: imconictranslation, imconicrotate
