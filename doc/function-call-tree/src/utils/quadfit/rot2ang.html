rot2ang

PURPOSE

Convert rotation matrix to Euler angles.

SYNOPSIS

function [phi,theta,psi] = rot2ang(A)

DESCRIPTION 

 Convert rotation matrix to Euler angles.
 Euler angles use the x-convention, i.e. phi, theta and psi are taken around the Z, X
 and again the Z-axis.

 See also: ang2rot
