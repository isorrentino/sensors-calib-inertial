◆ cadType2gain

Constant Property cadType2gain = containers.Map
protected

Accelerometer (raw fullscale to m/s^2 conversion): (fullscale in m/s^2)/(fullscale in raw) = (2*g)/(2^15) ~= 5.9855e-04 Gyroscope (raw fullscale to deg/s conversion): (fullscale in dps)/(fullscale in raw) = (250)/(2^15) ~= 7.6274e-03.

refer to http://wiki.icub.org/wiki/Distributed_Inertial_sensing
